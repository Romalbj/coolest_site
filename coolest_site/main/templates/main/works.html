MATH BOT

Telgram bot written in Python. It's based on the use of the requests library and the use of asynchronous functions. Thanks to the Wolframalfa's API, the bot solves equations and inequalities, and the use of the matplotlib library allows you to draw and display graphs of equations.

BOT TRANSLATOR

A Telgram bot written using VK API that allows you to translate text into a foreign language up to 10 Word pages. Due to API restrictions, translation into only 4 languages is possible, but for many this will be quite enough.

03

CASH REGISTER APP

An application written in Python with a GUI created in Tkinter. This application allows you to maintain simple cash registers, apply discounts and promotional codes and view your purchase history.
